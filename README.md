# combinatorics-calculator

Python tool to compute permutations (nPr) and combinations (nCr) using combinatorics formulas.

## Features

- Calculate all types of permutations (nPr)
- Calculate all types of combinations combinations (nCr)
- Factorial calculation


## Mathematical Formulas

Permutation:

nPr = n! / (n - r)!

Combination:

nCr = n! / (r! * (n - r)!)

## Example

Input:
n = 5  
r = 2  

Output:
Permutations (nPr) = 20  
Combinations (nCr) = 10  

## How to Run

Run the Python file:

```
combinatorics_calculator.py
```
